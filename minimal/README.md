# MINIMAL
Before we study complicated ORANI-G CGE model with using GEMPACK, we should study the simplied version first, which is MINIMAL - a slightly simplified CGE model designed for teaching purposes. 

MINIMAL is a percentage change model like ORANI-G, but has only 7 sectors, a simplified treatment of taxes, and no margins or multiproduction.
